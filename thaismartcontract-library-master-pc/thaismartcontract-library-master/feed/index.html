สร้างไฟล์ Nuget.config ใน Solution ของคุณ และวางข้อความต่อไปนี้ลงไป

 

<?xml version="1.0" encoding="utf-8"?>
<configuration>
  <packageSources>
    <add key="Thaismartcontract Repository" value="https://nuget.zarimpun.com/index.json" />
  </packageSources> 
</configuration>




จากนั้นไปที่เมนู Tools > Nuget Package Manager > Package Manger Console

แล้วพิมพ์ว่า Install-Package ตามด้วยชื่อแพ็คเกจ
